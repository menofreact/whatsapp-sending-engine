// turbo-all
1. Clone the engine:
   `git clone https://github.com/menofreact/whatsapp-sending-engine.git`
2. Change into the engine directory:
   `cd whatsapp-sending-engine`
3. Deploy the engine using Docker:
   `docker-compose up -d --build`
4. Use the API at `http://localhost:3000/api/send/direct` from other projects.
